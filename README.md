# Rock Paper Scissors

This repo showcases my learning of JavaScript fundamentals through the development of a Rock, Paper, Scissors game.

## Tech Stack

- Node.js 17+

## Installation

### 1. Repo Clone

```bash
git clone https://github.com/Sonam060703/Rock-Paper-Scissor.git
```


### 2. Run Game

```bash
node game.js
```
